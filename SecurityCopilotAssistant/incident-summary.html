Automatically analyze, summarize, and triage security incidents with AI-powered investigation

← Back to Home

Overview

The Incident Summary & Auto-Triage solution automatically processes new security incidents from Microsoft Sentinel, leveraging Security Copilot to provide instant AI-powered analysis. Reduce analyst workload by getting comprehensive incident summaries and triage recommendations within seconds of incident creation.

This solution includes 5 specialized Logic Apps, each handling a different aspect of incident analysis. Deploy them individually or together for a complete automated incident response workflow.
