Automatically enrich security entities with threat intelligence and contextual data using Security Copilot

← Back to Home

Overview

The Entity Enrichment solution automatically enhances security entities with comprehensive threat intelligence and contextual information from Microsoft Security Copilot. When triggered, the automation gathers data from multiple sources to provide a complete picture of any entity.

This solution includes 5 specialized Logic Apps, each optimized for a specific entity type. Deploy them individually based on your investigation needs, or deploy all at once for comprehensive entity enrichment capabilities.
