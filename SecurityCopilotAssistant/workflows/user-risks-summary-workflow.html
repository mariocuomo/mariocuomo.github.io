Automated weekly analysis of risky users with AI-powered investigation workflow

🎯 Workflow Overview

This Logic App runs every 7 days to perform an Entra User Risk Assessment. It queries Microsoft Sentinel for risk data, then uses Security Copilot to generate an executive summary report in HTML format.

📊 Sentinel KQL Queries ▶

Count By Risk State

AADRiskyUsers 
AADRiskyUsers
| summarize arg_max(TimeGenerated, *) by UserPrincipalName
| summarize count() by RiskState

Count By Risk Level

AADRiskyUsers 
AADRiskyUsers
| where RiskState == "atRisk"
| summarize arg_max(TimeGenerated, *) by UserPrincipalName
| summarize count() by RiskLevel

Risky User with Sign-in Evidence

AADRiskyUsers + AADUserRiskEvents 
AADRiskyUsers
| where RiskState == "atRisk"
| summarize arg_max(TimeGenerated, *) by UserPrincipalName
| project UserPrincipalName, RiskLevel
| join kind=leftouter (
    AADUserRiskEvents
    | where RiskState == "atRisk"
    | where Activity == "signin"
    | summarize arg_max(TimeGenerated, *) by UserPrincipalName
    | extend RiskReasons = iff(AdditionalInfo[0].Key == "riskReasons", 
                               AdditionalInfo[0].Value, RiskEventType) 
    | project UserPrincipalName, RiskReasons, IpAddress
) on UserPrincipalName
| project UserPrincipalName, RiskLevel, RiskReasons, IpAddress
🤖 Security Copilot Prompts ▶

Summary Report

Summarize the findings from this analysis into an executive report. Begin with a general overview. Include confidence and supporting evidence for the assessment. Below that, generate paragraph sections for a "Risk State distribution", "Risk Level distribution", and "Risky User with Sign-in risky evidence". It should be suitable for a less technical audience.

Convert to HTML

Convert the summary report into a well-formatted HTML email template with proper styling, headers, and sections ready for delivery.
